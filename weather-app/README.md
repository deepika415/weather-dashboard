# 🌤️ Weather Dashboard

A React.js weather dashboard with city search, current weather, and 5-day forecast.

## How to run

1. Extract this zip
2. Open the folder in VS Code
3. Open terminal and run:

npm install
npm run dev

4. Open http://localhost:5173 in your browser
5. Search: Hyderabad, Mumbai, Delhi, London, New York

## Tech Stack
React.js, Vite, Axios, CSS-in-JS
